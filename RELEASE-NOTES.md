# Notas de versión — v2.0.0 (2026-06-15)

## 📢 Novedades
- SpecFounder v2: sistema multi-agente con memoria persistente para construir la fundación del spec, tanto de código como de obras creativas.
- Generador de Visión, comandos `/iniciar` · `/retomar` · `/ayuda`, y estrategia de eficiencia de tokens.

## 🆙 Cómo actualizar
- `git pull` y revisa los cambios en [CHANGELOG.md](CHANGELOG.md).
- Si instalaste los comandos (`/iniciar`, `/retomar`, `/ayuda`), vuelve a copiarlos desde `specfounder-v2/comandos/` si cambiaron.
